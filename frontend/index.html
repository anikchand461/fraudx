// The Problem

What We Solve

Cybercriminals are faster than ever. Traditional security tools react too slowly. FraudX works in real time, at the browser level.

01 —
Phishing Attacks

Millions fall victim to fake login pages designed to steal credentials. FraudX identifies and flags deceptive domains before you type a single character.

02 —
Scam Websites

Fraudulent e-commerce sites and fake giveaways cost users billions annually. Our extension detects suspicious patterns and warns you instantly.

03 —
Malicious Links

A single click on a bad link can compromise your device. FraudX inspects every URL in real time before your browser loads the page.

// Architecture

How It Works

A lightweight extension talking to a powerful backend — zero latency protection on every page load.

01
URL Captured

The extension intercepts the current URL as soon as you navigate to any website, before the page fully renders.

02
Sent to Backend

The URL is securely forwarded to the FraudX FastAPI backend for deep analysis and classification.

03
AI Analysis

Pattern matching, heuristics, and threat intelligence databases assign a real-time risk score to the URL.

04
Alert Displayed

Results are returned instantly to the extension, which overlays a clear warning or safe signal directly on the page.

// Get Started

How to Set Up

Get FraudX running locally in under 5 minutes. Or skip setup entirely and install from the Chrome Web Store.

1

Clone the Repository

Pull the project from GitHub and navigate into the project folder.

2

Install Backend Dependencies

Navigate to /backend and run pip install -r requirements.txt to install all Python packages.

3

Start the FastAPI Server

Run uvicorn main:app --reload inside the backend folder. The API will boot on localhost:8000.

4

Load the Extension in Chrome

Open chrome://extensions, enable Developer Mode, click "Load Unpacked", and select the /extension folder.

$ git clone github.com/anikchand461/fraudx

# Install backend dependencies
$ cd fraudx/backend
$ pip install -r requirements.txt

# Start FastAPI server
$ uvicorn main:app --reload

# Load extension in Chrome
→ chrome://extensions
→ Enable Developer Mode
→ Load Unpacked → /extension
